# Andrej Karpathy

## Description
AI researcher and educator (ex-OpenAI, ex-Tesla AI Director), known for influential observations on LLM usage patterns — including the insight that knowledge manipulation in markdown is becoming a primary use case over code generation.

## Appearances
- [[sources/karpathy-method-claude-skills-obsidian]] — Cited as the originator of the "LLM as compiler and librarian" pattern; noted that a large fraction of his LLM usage has shifted to manipulating knowledge stored as markdown files
- [[sources/claude-obsidian-second-brain-defileo]] — @defileo's thread includes and attributes the full Karpathy LLM Wiki prompt; spreading the pattern via copy-paste distribution
