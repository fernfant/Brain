# Obsidian

## Description
Free, local-first markdown-based knowledge editor used as the IDE for browsing and navigating an LLM-maintained wiki — featuring backlinks, graph view, Dataview queries, and Marp slide rendering.

## Appearances
- [[sources/karpathy-method-claude-skills-obsidian]] — Recommended as the primary interface for the LLM Wiki system; key features: graph view (spot knowledge clusters and gaps), backlinks, Dataview plugin, Marp for presentations
