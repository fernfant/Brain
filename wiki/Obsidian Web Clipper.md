# Obsidian Web Clipper

## Description
Browser extension that converts any web page into clean markdown and saves it directly into an Obsidian vault with URL, title, date, and tags — the recommended capture tool for the LLM Wiki system.

## Appearances
- [[sources/karpathy-method-claude-skills-obsidian]] — Recommended as the primary capture mechanism; saves to raw/ automatically; companion hotkey downloads images locally for LLM reference
