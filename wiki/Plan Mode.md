# Plan Mode

## Description
A read-only mode in Claude Code (activated via Shift+Tab) where Claude analyses the codebase and develops an implementation strategy before making any changes — reduces debugging cycles and prevents architectural misdirection.

## Appearances
- [[sources/leveraging-claude-code-senior-engineers-guide]] — activate before writing code to prevent misdirection; reduces debugging cycles
- [[sources/claude-code-tutorial-beginners-2026]] — use for tasks touching 2+ files or involving architectural decisions; activated via Shift+Tab
- [[sources/power-users-migrate-claude-productivity]] — start in Plan Mode (Shift+Tab) for better one-shot implementation
